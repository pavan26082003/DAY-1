HTML <iframe> ‚Äì Complete Notes
‚úÖ 1. What is an <iframe>?
<iframe> stands for inline frame.

It is used to embed another HTML page inside the current page.

It creates a window within your page to show external or internal content.

‚úÖ 2. Basic Syntax
html
Copy code
<iframe src="https://example.com" width="600" height="400" title="Example Site"></iframe>
src: URL of the page to embed.

width and height: Size of the iframe.

title: Accessibility attribute describing the iframe content.

‚úÖ 3. Common Attributes
Attribute	Purpose
src	URL of the document to display inside the iframe
width	Width of the iframe (pixels or %)
height	Height of the iframe
title	Description for screen readers (important for accessibility)
name	Name used for targeting the iframe with links or scripts
frameborder	Whether to show a border around the iframe (0 or 1)
allow	Permissions granted inside iframe (e.g., camera, microphone)
sandbox	Adds restrictions on what the iframe content can do
loading	lazy or eager loading behavior
referrerpolicy	Controls referrer information sent with requests

‚úÖ 4. Important Attributes Explained
üîπ sandbox
Applies extra restrictions to iframe content for security.

Examples:

sandbox="allow-scripts" ‚Äî allows scripts inside iframe.

sandbox="allow-same-origin" ‚Äî treats content as same origin.

If no sandbox attribute, iframe is fully enabled.

üîπ allow
Used to grant permissions, e.g., allow="camera; microphone; fullscreen"

Controls features the iframe content can access.

üîπ loading
Controls lazy loading.

loading="lazy" defers loading iframe until it's near viewport, improving performance.

‚úÖ 5. Use Cases
Use Case	Explanation
Embedding YouTube videos	Embed a video player inside your webpage
Google Maps integration	Show maps with zoom and location features
Embedding external websites	Display another site‚Äôs content without leaving page
Ads and widgets	Show advertisements or interactive widgets

‚úÖ 6. Example

Copy code

<iframe 
  width="560" 
  height="315" 
  src="https://www.youtube.com/embed/dQw4w9WgXcQ"

‚úÖ 7. Accessibility Tips
Always add a descriptive title attribute.

Use aria attributes if needed.

Avoid auto-playing media in iframe for better user experience.

‚úÖ 8. Security Considerations
Use sandbox attribute to restrict iframe content.

Avoid embedding untrusted sources to prevent clickjacking or malicious scripts.

Use CSP (Content Security Policy) headers to control iframe usage.

Alternatives to <iframe> for embedding content
1. <object> tag
Can embed HTML pages, PDFs, images, videos, and more.

Similar to iframe but slightly different behavior and support.

html
Copy code
<object data="https://example.com" width="600" height="400"></object>
2. <embed> tag
Primarily for embedding multimedia like PDFs, videos, or flash content.

Less flexible for full web pages compared to iframe.

html
Copy code
<embed src="https://example.com" width="600" height="400" />
3. AJAX / Fetch + Dynamic Content Injection
Use JavaScript to fetch external content (HTML or data) and inject it into your page‚Äôs DOM.

Offers better control but can be limited by CORS (Cross-Origin Resource Sharing) restrictions.

js
Copy code
fetch('https://example.com/somepage.html')
  .then(response => response.text())
  .then(html => {
    document.getElementById('container').innerHTML = html;
  });
